# Card
### Flexible and extensible content container with multiple variants and options

## Example
Below is an example of a basic card with mixed content and a fixed width.

Cards have no fixed width to start, so they’ll naturally fill the full width of its parent element
unless specified explicitly like in this example.

<Preview>
  <BasicCard />
</Preview>

```typescript jsx
<Card width="18rem">
  <CardImg src={src} alt="swiss-landscape" />
  <CardBody>
    <Heading size="h5">Card Title</Heading>
    <p>
      Some quick example text to build on the card title and make up the bulk of the card's
      content.
    </p>
    <Button>Go Somewhere</Button>
  </CardBody>
</Card>
```

## Content types
Cards support a wide variety of content, including images, text, list groups, links, and more.

<InfoCard>
  <div>
    It maybe worth noting that most of the card subcomponents exported by Stylestrap are just
    simple extensions to their default form. This means that you are
    not bound to use these subcomponents. You have full flexibility on what you want to
    render inside a card.
    <br />
    <br />
    For example, <code>{` <CardTitle /> `}</code> is just a <code>{` <Heading /> `}</code> with
    <code> size="h5" </code> and <code> margin-bottom=0.75rem</code>.
  </div>
</InfoCard>

Use it whenever you need a padded section within a card.

### Titles, text, and links

Stylestrap comes bundled with a set of Card-* helper components
which make styling inside cards pretty easy. All text based components
extend from Stylestrap Typography components, which means all base
props are supported by these helper components as well.

### Images
Place images on top, bottom or in the middle of a Card.

<Preview>
  <CardImagePreview />
</Preview>

```typescript jsx
import src from './cardImg.jpg';
//...
<Grid columns="3">
  <Card width="15rem">
    <CardImg src={src} />
    <CardBody>
      <CardText>Some text description of the image below the image</CardText>
    </CardBody>
  </Card>

  <Card width="15rem">
    <CardBody>
      <CardText>Some text description of the image above the image</CardText>
    </CardBody>
    <CardImg src={src} />
  </Card>

  <Card width="15rem">
    <CardBody>
      <CardText>Some text above the image</CardText>
    </CardBody>
    <CardImg src={src} />
    <CardBody>
      <CardText>Some text below the image</CardText>
    </CardBody>
  </Card>
</Grid>
```

### List Groups
Create lists of content in a card with a flush list group. The flush prop is
passed to ignore the default left and right borders of the ListGroup.

<Preview>
  <ListGroupPreset />
</Preview>

```typescript jsx
<Card width="18rem">
  <CardHeader>Optional Header</CardHeader>
  <ListGroup flush>
    <ListGroupItem>Cras justo odio</ListGroupItem>
    <ListGroupItem>Dapibus ac facilisis in</ListGroupItem>
    <ListGroupItem>Vestibulum at eros</ListGroupItem>
  </ListGroup>
</Card>
```

### Header and Footer
Add an optional **header** and/or **footer** within a card.

<Preview>
  <OptionalHeader />
</Preview>

```typescript jsx
<Card>
  <CardHeader>Optional Header</CardHeader>
  <CardBody>
    <CardTitle>Special title treatment</CardTitle>
    <CardText>With supporting text below as a natural lead-in to additional content.</CardText>
    <Button>Go Somewhere</Button>
  </CardBody>
  <CardFooter>Optional Footer</CardFooter>
</Card>
```

You can use the StyledComponents' `as` prop to change the Header component to use
any tag or component you like. Below you see how we can change the tag to an "h5".
Similarly, you can change it to use the stylestrap `Heading` component and make
use of all its props.

<Preview>
  <OptionalHeaderBold />
</Preview>

```typescript jsx
<Card>
  <CardHeader as="h5">Optional Header</CardHeader>
  <CardBody>
    <CardTitle>Special title treatment</CardTitle>
    <CardText>With supporting text below as a natural lead-in to additional content.</CardText>
    <Button>Go Somewhere</Button>
  </CardBody>
</Card>
```

### Text Align
Align contents to left, right, center or justify using the `textAlign` prop.

<Preview>
  <TextAlign />
</Preview>

```typescript jsx
<Card textAlign="justify" width="16rem">
  <CardBody>
    <CardTitle>Title</CardTitle>
    <CardText>
      With some supporting text justified to make a solid sense of order and standard. like
      a newspaper column.
    </CardText>
    <Button>Button</Button>
  </CardBody>
</Card>
<Card textAlign="center" width="16rem">
/* Similar content as above */
</Card>
<Card textAlign="right" width="16rem">
/* Similar content as above */
</Card>
```
