# Badge
<Helpers.Subtitle>Our small count and labeling component.</Helpers.Subtitle>

## Example
Badges scale to match the size of the immediate parent element by using relative font sizing and em units.

  <Heading size="h1">Example heading <Badge color="secondary">New</Badge></Heading>
  <Heading size="h2">Example heading <Badge color="secondary">New</Badge></Heading>

  <Heading size="h4">Example heading <Badge color="secondary">New</Badge></Heading>
  <Heading size="h5">Example heading <Badge color="secondary">New</Badge></Heading>
  <Heading size="h6">Example heading <Badge color="secondary">New</Badge></Heading>
</Preview.Adaptive>

```jsx
<Heading size="h1">Example heading <Badge color="secondary">New</Badge></Heading>
```

Badges can also be used as part of links or buttons.

## Link Badges

You can make the badges actionable just by passing an `href` prop.
The HTML tag will be changed to the anchor tag (unless you override it with the as prop).

<Preview>
  <Spaced>
    <Badge href="#">Primary</Badge>
    <Badge href="#" color="secondary">Secondary</Badge>
    <Badge href="#" color="success">Success</Badge>
    <Badge href="#" color="warning">Warning</Badge>
    <Badge href="#" color="danger">Danger</Badge>
    <Badge href="#" color="info">Info</Badge>
    <Badge href="#" color="dark">Dark</Badge>
    <Badge href="#" color="light">Light</Badge>
  </Spaced>
</Preview>


```jsx
<Badge href="#">Primary</Badge>
<Badge href="#" color="secondary">Secondary</Badge>
<Badge href="#" color="success">Success</Badge>
```
