# Button
### For actions in forms, dialogs, and more with default support for multiple sizes, variants, and more.

<PropsTable of={Button} />

## Examples
Stylestrap includes several predefined button styles, each serving its own semantic purpose.

Note that unlike in raw bootstrap, we don't include the "link" style in our buttons.
Instead, we have dedicated Link and Anchor components, which are more flexible
and easier to extend.

<Preview>
  <Grid columns="8" gap="xxs">
    <Button>Primary</Button>
    <Button color="secondary">Secondary</Button>
    <Button color="success">Success</Button>
    <Button color="warning">Warning</Button>
    <Button color="danger">Danger</Button>
    <Button color="info">Info</Button>
    <Button color="dark">Dark</Button>
    <Button color="light">Light</Button>
  </Grid>
</Preview>

```typescript jsx
<Button>Primary</Button> <!-- You can pass color="primary" explicitly as well -->
<Button color="secondary">Secondary</Button>
<Button color="success">Success</Button>
<Button color="warning">Warning</Button>
<Button color="danger">Danger</Button>
<Button color="info">Info</Button>
<Button color="dark">Dark</Button>
<Button color="light">Light</Button>
```

>Using color to add meaning only provides a visual indication,
which will not be conveyed to users of assistive technologies â€“ such as screen readers.
Ensure that information denoted by the color is either obvious from the content itself (e.g. the visible text),
or is included through alternative means, such as additional text hidden with the ScreenReader component.

## Button tags
The buttons by default uses the html button tag. However, just like any other stylestrap component,
can be customized to use other tags like `input` or `a` using the styled-components' `as` prop.

<Preview>
  <Spaced>
    <Button as="a">Primary</Button>
    <Button as="input" type="button" value="Input" />
    <Button as="input" type="submit" value="Submit" />
    <Button as="input" type="reset" value="Reset" />
  </Spaced>
</Preview>

```typescript jsx
<Button as="a">Primary</Button>
<Button as="input" type="button" value="Input" />
<Button as="input" type="submit" value="Submit" />
<Button as="input" type="reset" value="Reset" />
```

## Button Flavors
Stylestrap includes few other variants of Buttons which can be changed using
the `flavor` prop.

Note that contextual variants like `color="light"` will work best with dark backgrounds.
